HTML Tags Video, Audio and Canvas

HTML5 allows us to integrate video, audio and graphics using the canvas tag.

Video tags can have different attributes, such as controls (to show the controls), autoplay, poster (to show an image until video starts) or loop (to keep playing the video).

Audio tags also have attributes such as controls, autoplay, lopp and muted.

The canvas tag is used to create graphics with JavaScript (although other languages can be used too).
